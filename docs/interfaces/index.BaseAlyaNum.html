Interface BaseAlyaNum

The base version of an AlyaNum object, stripped of its metatables and metamethods. This type commonly appears when sending AlyaNum objects over the client-server boundary or saving it in datastores.

Example

{sign: -1, multiplicand: 4, exponent: 2, tetrate: 1, pentate: 0, hexate: 0, heptate: 0}
// This represents -1 * 10 ^^ (10 ^ (10 ^ 4)).
interface BaseAlyaNum {
    exponent: number;
    heptate: number;
    hexate: number;
    multiplicand: number;
    pentate: number;
    sign: -1 | 0 | 1;
    tetrate: number;
}
